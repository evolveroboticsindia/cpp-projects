# 🧮 Simple Calculator in C++

A console-based Simple Calculator developed using C++ that performs basic arithmetic operations through an interactive menu-driven interface.

The project allows users to perform operations such as:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
with a colorful console UI for better user experience.


# 📌 Project Overview

This project is designed to demonstrate the fundamentals of C++ programming by implementing a simple calculator application.

The calculator accepts user input, performs mathematical operations, and displays the result instantly through a menu-driven system.

The main objective of this project is to understand:
- arithmetic operations
- switch-case statements
- loops
- functions
- console-based application development

# 📌 Functionalities

## ➕ Addition
Adds two numbers and displays the result.

## ➖ Subtraction
Subtracts the second number from the first number.

## ✖ Multiplication
Multiplies two numbers and displays the result.

## ➗ Division
Divides the first number by the second number.

The program also checks:
- division by zero
- invalid operations

## 🔢 Modulus
Calculates the remainder of division between two integers.
